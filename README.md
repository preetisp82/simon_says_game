# Simon Says Game 🎮

A fun and interactive Simon Says memory game built using HTML, CSS, and JavaScript.

## 🛠️ Features
- Classic Simon Says memory game
- Color and sound-based pattern recognition
- Increasing difficulty with each round
- Score tracking

## 💻 Tech Stack
- HTML
- CSS
- JavaScript
